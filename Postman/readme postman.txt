В папке "Postman" находятся файлы,в которых я пробно тестирую API согласно тестовым заданиям к системе Users https://testbase.atlassian.net/wiki/spaces/USERS/overview 
Cама система Users http://users.bugred.ru

Файл "UserTest" содержит REST-запросы для:
- регистрации пользователя в системе;
- создания нового пользователя;
- создания новой компании;
- создания задач для пользователя;
- добавления и удаления аватара;
- поиск с различными статусами ответов.

Файл "Тестовые случаи по Users" содержит Тест-кейсы для проверки корректности работы системы Users.
